About Simple Blog Application

The Simple Blog Application is a web-based platform that allows users to create, read, update, and delete blog posts. It includes features such as user authentication, post management, and comment functionality.

User Authentication

Secure user registration and login with unique usernames and passwords.

Post Management

Create, edit, and delete your own blog posts with ease.

Comment Functionality

Engage with other users by commenting on their blog posts.

User Story: View Blog Posts

As a guest user, I want to view blog posts so that I can read the content.

Acceptance Criteria

  • The system shall display a list of blog posts on the homepage.
  • Each blog post shall display the title, author, and publication date.
  • The system shall allow users to click on a blog post to view its full content.

Mapped Functional Requirement

FR3: The system shall allow registered users to create, read, update, and delete their own blog posts.
